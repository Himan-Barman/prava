# Auth Flow

Implemented:

- OTP request and verification.
- Signup through verified email flow.
- Login with JWT access token.
- Hashed refresh tokens.
- Refresh-token rotation.
- Logout and logout-all.
- Session listing and session revocation.
- Password reset OTP.

V1 endpoints are exposed under `/api/v1/auth/*`; legacy compatibility remains under `/api/auth/*`.
